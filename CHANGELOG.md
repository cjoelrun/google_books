CHANGELOG
=========

v0.0.1
------
* Initial Version, handling the following attributes:
  * `title` - *String*
  * `authors` - *Array*
  * `publisher` - *String*
  * `published_date` - *Date*
  * `isbn` - *String*
  * `isbn_10` - *String*
  * `page_count` - *FixNum*
  * `categories` - *Array*
  * `description` - *String*
  * `average_rating` - *Float*
  * `ratings_count` - *FixNum*
  * `covers` - *Hash*, with the following keys:
      * `:thumbnail`
      * `:small`
      * `:medium`
      * `:large`
      * `:extra_large`
  * `preview_link` - *String*
  * `info_link` - *String*
